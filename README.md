# docusaurus

docusaurus 是一个为开源项目创建一个静态网站的工具，由 facebook 开发。使用 Markdown 即可更新网站。文档、API 一应俱全

docusaurus 依赖于 node 环境，在 npm 上有相应的包。通过 npx docusaurus-init 进行网站初始化搭建，
